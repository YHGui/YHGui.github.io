Markdown 快速入门学习

段落、标题、区块代码

一个段落是由一个以上的连接的行句组成，而一个以上的空行则会划分出不同的段落（空行的定义是显示上看起来像是空行，
就被视为空行，例如有一行只有空白和tab，那该行也会被视为空行），一般的段落不需要用空白或换行缩进。

Markdown支持两种标题语法，Setext和atx形式。Setext形式是用底线的形式，利用=表示最高阶标题，而-表示第二阶标题，
Atx形式则在行首插入1到6个#对应1至6阶标题。
区块引用则使用email形式的‘>’角括号，使用‘>’得到如下效果：

This is a blockquote.

This is the second paragraph in the blockquote

This is an H2 in a blockquote.

当出现空白行的时候，则表示结束引用。

修辞和强调

Markdown 使用星号（）和下划线”_”来标记需要强调的区段，例如：
Some of these words are emphasize.
Some of these words are emphasize.
Use two asterisks for *strong emphasize.
Or,if you prefer, use two underscores instead.

列表

无序列表使用星号、加号和减号作为列表的项目来标记，这些符号都是可以使用的，例如：

  • Candy
  • Gum
  • Booze
  • Candy
  • Gum
  • Booze
  • Candy
  • Gum
  • Booze

有序列表则是使用一般的数字接着一个英文句点作为项目标记：

  1. Red
  2. Green
  3. Blue

  • A list item

    With multiple paragraphs.

  • Another item in the list

链接

Markdown 支持两种形式的链接语法：行内和参考两种形式，两种都是使用角括号来把文字转成连接。
行内形式是直接在后面用括号直接接上链接：
This is an example link.
This is an example link.

参考形式的链接让你可以为链接定一个名称，之后你可以在文件的其他地方定义该链接的内容：
I get 10 times more traffic from Google than from
Yahoo or MSN.

参考形式的链接之间不能有其他的语句，其中title是可选的。
而图片的链接与一般的连接类似。

代码

在一般的段落文字中，你可以使用反引号 来标记代码区段，区段内的 &、< 和 > 都会被自动的转换成 HTML 实体，
这项特性让你可以很容易的在代码区段内插入 HTML 码：
I strongly recommend against using any <blink> tags.

I wish SmartyPants used named entities like &mdash;
instead of decimal-encoded entites like &#8212;.

如果要建立一个已经格式化好的代码区块，只要每行都缩进 4 个空格或是一个 tab 就可以了，而 &、< 和 > 也一样会自动转成 HTML 实体。

Markdown 语法:

If you want your page to validate under XHTML 1.0 Strict,
you’ve got to put paragraph tags in your blockquotes:

For example.



输出 HTML 为：

If you want your page to validate under XHTML 1.0 Strict,
you’ve got to put paragraph tags in your blockquotes:

<blockquote>
<p>For example.</p>
</blockquote>
